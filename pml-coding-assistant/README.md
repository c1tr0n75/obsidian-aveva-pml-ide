# PML Coding Assistant

This folder is the first core product area of the skill.

It contains the references, production examples, reusable PML patterns, and validation scripts used to generate, review, debug, and explain AVEVA PML/PML2 with near-zero syntax error.

## LLM Navigation

1. Start with [../SKILL.md](../SKILL.md) for the global coding contract.
2. Use [references/object-type-index.md](references/object-type-index.md) for PML object, gadget, and method lookup.
3. Use [references/database/database-data-model-index.md](references/database/database-data-model-index.md) before creating, querying, or modifying E3D database elements.
4. Use [examples/pml-patterns/](examples/pml-patterns/README.md) for reusable, runnable PML/PML2 templates.
5. Use [production-reference/](production-reference/) to ground syntax in real production code and real DBOUTPUT/DBLISTING files.
6. Use [scripts/](scripts/) for local validation and PML example linting.
