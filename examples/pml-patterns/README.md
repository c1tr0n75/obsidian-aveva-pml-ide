# PML Patterns

This folder contains small, reusable PML/PML2 patterns for new code generation and review.

Use it for:
- Complete `.pmlfnc`, `.pmlobj`, `.pmlfrm`, and legacy `.mac` examples.
- Focused templates that demonstrate a verified PML idiom.
- Small companion outputs, such as `.txt` or `.csv`, when they document the expected result of a pattern.

Patterns should be runnable, lowercase where file naming conventions apply, and grounded in the references listed from [SKILL.md](../../SKILL.md).
