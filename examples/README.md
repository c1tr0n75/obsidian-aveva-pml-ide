# Examples

This folder stores reusable and generated artifacts that help the agent work faster without treating past conversations as memory.

- [pml-patterns/](pml-patterns/README.md) contains runnable PML/PML2 templates and small companion outputs that document expected results.
- [kpi-reports/](kpi-reports/README.md) contains generated HTML KPI dashboards and interactive reports produced on demand from AVEVA E3D project or database context.

Long-lived project standards, engineering practices, naming rules, BOM guidance, and EPC specifications belong in [project-knowledge/](../project-knowledge/README.md), not in this folder.
