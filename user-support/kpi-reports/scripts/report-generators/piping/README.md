# Piping Report Generators

This folder contains repeatable generators for piping KPI report packages.

## Demo Packages

The bundled demo dashboards are generated by:

```bash
node user-support/kpi-reports/scripts/generate_demo_kpi_reports.js
```

## Before/After Diff

Generate a piping change report from two DB output listings:

```bash
node user-support/kpi-reports/scripts/report-generators/piping/compare_dboutput_piping.js before.txt after.txt user-support/kpi-reports/piping-diff-demo
```

The output folder follows the KPI report package standard.
