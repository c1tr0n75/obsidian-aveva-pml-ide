# Report Generators

This folder contains repeatable report-generation scripts.

Generators should:
- Read explicit source files.
- Produce a self-contained package under [kpi-reports/](../../README.md).
- Include `README.md`, `report.json`, HTML, JSON data, `summary.md`, and CSV exports.
- Update report indexes when producing bundled/demo reports.

## Current Generators

- [piping/](piping/README.md) - demo and comparison generators for piping DB output listings.
