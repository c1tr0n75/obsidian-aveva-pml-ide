# KPI References

This folder defines reusable KPI meanings, formulas, source fields, and caveats for AVEVA E3D dashboards.

Use these pages when generating or reviewing reports under [kpi-reports/](../README.md). KPI definitions should make dashboards auditable: an engineer should be able to see what was counted, where it came from, and what the limitation is.

- [kpi-definitions.md](kpi-definitions.md) — standard KPI dictionary for model health, MTO readiness, construction readiness, spatial density, and route complexity.
