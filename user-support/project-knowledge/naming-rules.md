# Naming Rules

Use this page to record project naming conventions that generated PML, dashboards, and QA checks should respect.

## Piping

Recommended starter pattern:

```text
<nominal-bore>-<service-code>-<sequence>
```

Example:

```text
100-B-1
```

## Supports

Recommended starter pattern:

```text
PS-<area>-<sequence>
```

Example:

```text
PS-01-00107
```

## Dashboard Use

KPI reports may flag names that do not match the active project pattern. When a report flags naming issues, link the issue back to this page.
