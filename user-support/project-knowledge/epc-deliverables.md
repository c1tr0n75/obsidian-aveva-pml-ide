# EPC Deliverables

Use this page to map generated reports and PML patterns to EPC project deliverables.

## Starter Mapping

| Deliverable | Useful Vault Output |
| --- | --- |
| Model health review | `model-health` KPI reports |
| Construction readiness review | `mto-readiness` KPI reports |
| Material take-off review | CSV exports from KPI packages |
| Engineering rules clarification | Pages in `user-support/project-knowledge/` |
| Automation template | PML patterns under `pml-coding-assistant/examples/pml-patterns/` |

## Review Rhythm

- Use model health reports before discipline review meetings.
- Use MTO readiness reports before material extraction or procurement review.
- Use diff reports for weekly progress and change discussions.
